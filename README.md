-   Background: #EFECE3

-   Detail: #8FABD4
-   Accent: #4A70A9
-   Success: #73AF6F
-   Danger: #DC0E0E

## How to run

```
npm install
```

Create env.local at root and add

```
JWT_SECRET=<Your jwt secret>
```

Run

```
npm run dev
```

Open
http://localhost:3000
